# Zsh Key Bindings Reference

## Navigation
- **↑ / ↓** - History substring search (filters based on what you've typed)
- **Ctrl + R** - FZF history search (interactive fuzzy finder)
- **Ctrl + T** - FZF file search (insert file path)
- **Alt + C** - FZF directory search (change directory)
- **Ctrl + ←/→** - Jump word backward/forward
- **Home / End** - Jump to beginning/end of line

## Editing
- **Ctrl + Backspace** - Delete word backward
- **Ctrl + Delete** - Delete word forward
- **Alt + Backspace** - Delete word backward
- **Ctrl + X, Ctrl + E** - Edit command in $EDITOR

## Completion & Suggestions
- **Tab** - Trigger completion menu
- **→** - Accept autosuggestion
- **Ctrl + Space** - Accept autosuggestion (alternative)
- **Shift + Tab** - Navigate completion menu backward

## History
- **Ctrl + R** - Search history with FZF
- **↑ / ↓** - Search history by substring

## Other
- **Ctrl + C** - Cancel current command
- **Ctrl + D** - Exit shell
- **Ctrl + L** - Clear screen
- **Ctrl + Z** - Suspend process (use `fg` to resume)

## Directory Navigation
- Type directory name without `cd` (auto_cd enabled)
- **..** - Go up one directory
- **...** - Go up two directories
- **....** - Go up three directories
